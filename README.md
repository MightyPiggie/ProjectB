Doel van project:
Het doel van het project is het voorzien van gebruikers inzicht in hun eigen gaming gedrag, en ook die van hun vrienden. Hierbij is het van belang dat er ook uit verschillende klantsegmenten data te zien valt zoals graphieken.

Hoe open je de GUI:
Open de FA AI.py bestand met een IDE die bedoeld is voor Python (Bijvoorbeeld Pycharm). Zorg ervoor dat de steam.json bestand in dezelfde folder zit als de FA AI.py bestand. Als het bestand geopend is druk je op de run knop en de GUI zal geopend worden.
